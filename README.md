# Interactive E-Commerce Revenue Forecasting App

An interactive, data-driven web dashboard built using **Streamlit** and a **Random Forest Regressor** backend to forecast future daily sales revenue based on operational marketing metrics.

## 🚀 App Features
* **Live Forecasting Machine:** Use the sidebar sliders to input ad-spend and traffic metrics to generate real-time revenue predictions.
* **Model Validation Plots:** Dynamic regression chart demonstrating predictive accuracy.
* **Accuracy:** R2 Score of ~95%.
